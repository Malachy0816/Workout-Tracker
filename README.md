 # 🏋️ Workout Tracker PWA

A Progressive Web Application (PWA) built using Ionic and Angular that allows users to browse exercises, log workouts, and track their fitness history.

This project demonstrates the use of Angular standalone components, external API integration, and mobile-first design principles. It is fully deployable and installable as a mobile app.

---

## 🚀 Features

*  External API integration for exercise data
*  Real-time search functionality to quickly find exercises
*  Log workouts with custom inputs
*  View workout history
*  Persistent local storage using Ionic Storage
*  Settings page for managing app data
*  Installable as a Progressive Web App (PWA)

---

## 🛠️ Technologies used

* Ionic
* Angular
* TypeScript
* Firebase Hosting
* Ionic Storage
* REST API integration

---

## 🧑‍💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/Malachy0816/Workout-Tracker
cd workout-tracker
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
ionic serve
```

The application will open in your browser at:
http://localhost:8100
